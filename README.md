Open the folder in a terminal and run: <code>npm install</code>

If you don't have a file named <b>.babelrc</b> create one and copy the following code into it

    {
    "presets": [
        [
        "env",
        {
            "esmodules": false,
            "targets": {
            "browsers": [
                "> 3%"
            ]
            }
        }
        ]
    ],
    "plugins": [
        "transform-runtime"
    ]
    }
